# agrinho20026
Agroforte: Futuro Sustentável é um projeto web interativo desenvolvido com HTML, CSS e JavaScript que promove a conscientização sobre sustentabilidade na agricultura. O site conta com recursos como modo escuro, leitura de texto por voz e uma calculadora de árvores plantadas, incentivando a reflexão sobre o impacto ambiental e o futuro sustentável.
